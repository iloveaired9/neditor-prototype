# CHEditor 플러그인 분석 보고서

이 문서는 뽐뿌 플랫폼에서 사용 중인 `cheditor` 구현의 플러그인 구조와 주요 기능을 분석한 결과입니다.

## 핵심 기능 및 플러그인

### 1. 기본 서식 (ToolbarPlugin 해당)
- **텍스트 스타일**: 굵게(Bold), 기울기(Italic), 밑줄(Underline), 취소선(Strikethrough), 위 첨자(Superscript), 아래 첨자(Subscript).
- **색상 설정**: 글자색(ForeColor), 배경색/형광펜(BackColor).
- **타이포그래피**: 글꼴(FontName), 글자 크기(FontSize), 줄 간격(LineHeight).
- **정리 기능**: 서식 지우기(RemoveFormat - 인라인 스타일 제거), 태그 제거(ClearTag - 모든 HTML 태그 제거).

### 2. 레이아웃 및 단락
- **정렬**: 왼쪽, 가운데, 오른쪽, 양쪽 정렬.
- **목록**: 번호 매기기(Ordered List), 글머리 기호(Unordered List).
- **들여쓰기**: 들여쓰기 늘리기(Indent), 줄이기(Outdent).
- **구조화**: 단락 스타일(FormatBlock - 제목 1-6, 본문 등).

### 3. 삽입 및 객체
- **하이퍼링크**: 링크 삽입/편집, 링크 해제.
- **표(Table)**: 표 생성, 크기 조절, 셀 병합/분할 및 속성 수정을 포함한 종합 도구.
- **특수 문자**: 수학 기호 및 고전 기호를 위한 기호 선택기(Symbol).
- **이모티콘**: 표정 아이콘 선택기(`popup/icon.html`).
- **글 상자**: 강조 섹션을 위한 스타일이 적용된 div 컨테이너(TextBlock).
- **가로줄**: 구분선 삽입(HR).

### 4. 미디어 및 업로드
- **이미지 업로드**: `popup/image.html5.html`을 통한 멀티 파일 업로더. 드래그 앤 드롭 및 뽐뿌 전용 서버 연동 지원.
- **웹 이미지**: 외부 이미지 URL 직접 삽입.
- **임베드**: 플래시 및 미디어(유튜브 등 외부 서비스의 소스 코드 삽입 도구).
- **지도**: 구글 지도 연동.

### 5. 특화 기능 (CHEditor Ppom)
- **파일 동기화**: 에디터 본문의 이미지와 하단 첨부 파일 리스트 간의 실시간 동기화.
- **대표 이미지**: 업로드된 이미지 중 특정 이미지를 게시글 썸네일로 지정하는 로직.
- **자동 스크랩**: URL 붙여넣기 시 메타데이터를 추출하여 요약 카드를 생성하는 `get_meta_make_thumb` 로직 (Neditor의 ScrapPlugin 전신).

## 기술적 구현 특징
- **구조**: UI 정의를 위해 XML 기반 템플릿 시스템(`template.xml`) 사용.
- **엔진**: Iframe 기반의 편집 영역을 사용하며, `document.execCommand`와 수동 DOM 조작에 크게 의존.
- **대화상자**: 팝업 방식의 윈도우(`popup/*.html`)를 사용하여 메인 에디터와 `window.opener`를 통해 통신.
