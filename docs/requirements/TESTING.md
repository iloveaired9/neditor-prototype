# SmartImageEditorPlugin 테스트 가이드

## 🧪 테스트 환경 설정

### 1. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 2. 프로덕션 빌드
```bash
npm run build
npm run preview
```

## ✅ 기능 테스트 체크리스트

### 1. 기본 동작

- [ ] **플러그인 로드**
  - 콘솔에 "SmartImageEditorPlugin initializing..." 메시지 표시
  - 에러 없음

- [ ] **모달 생성**
  - 모달 HTML 요소 생성 확인
  - CSS 스타일 정상 로드

### 2. 이미지 편집 열기

- [ ] **ImageElement 로드**
  ```javascript
  // 콘솔에서 테스트
  const editor = window.editorInstance;
  const img = new Image();
  img.src = 'data:image/png;base64,...';
  await editor.plugins.smartImageEditor.openEditor(img);
  ```

- [ ] **모달 표시**
  - 모달이 화면에 나타남
  - 원본 이미지가 Canvas에 표시됨

### 3. Crop (자르기) 기능

#### 테스트 케이스 1: 자유 자르기
- [ ] 종횡비 "자유" 선택
- [ ] X, Y, 너비, 높이 값 입력
  ```
  X: 50, Y: 50, 너비: 100, 높이: 100
  ```
- [ ] "적용" 버튼 클릭
- [ ] Canvas 크기가 변경됨 (100x100)

#### 테스트 케이스 2: 비율 자르기
- [ ] 종횡비 "16:9" 선택
- [ ] 적용
- [ ] Canvas 크기가 16:9 비율로 조정됨

### 4. Rotate (회전) 기능

#### 테스트 케이스 1: 자유 회전
- [ ] 회전 각도 슬라이더 조정 (45도)
- [ ] 각도 값 표시 정상
- [ ] "적용" 버튼 클릭
- [ ] 이미지가 45도 회전됨

#### 테스트 케이스 2: 빠른 회전
- [ ] "+90°" 버튼 클릭
- [ ] 이미지가 90도 회전됨
- [ ] Canvas 크기가 조정됨

#### 테스트 케이스 3: 배경색 설정
- [ ] 배경색 선택 (빨강)
- [ ] 회전 적용
- [ ] 회전 후 빈 공간이 선택한 색으로 채워짐

### 5. Filter (필터) 기능

#### 테스트 각 필터별
- [ ] **Grayscale** - 흑백 변환
- [ ] **Sepia** - 갈색톤 적용
- [ ] **Blur** - 흐릿함 적용
- [ ] **Invert** - 색상 반전
- [ ] **Brightness** - 밝아짐
- [ ] **Contrast** - 명암 증가
- [ ] **Saturate** - 채도 증가
- [ ] **Hue-Rotate** - 색상 회전
- [ ] **Vintage** - 빈티지 효과
- [ ] **Cool** - 차가운 톤
- [ ] **Warm** - 따뜻한 톤
- [ ] **Charcoal** - 연탄 효과
- [ ] **Sketch** - 스케치 효과
- [ ] **Posterize** - 포스터 효과
- [ ] **Cartoon** - 만화 효과

각 필터에 대해:
1. 필터 버튼 클릭 (활성화)
2. 강도 슬라이더 조정 (0-100%)
3. "적용" 버튼 클릭
4. 필터 효과 확인

### 6. Adjust (보정) 기능

#### 테스트 케이스 1: 밝기
- [ ] 밝기 슬라이더 조정 (+50)
- [ ] "적용" 버튼 클릭
- [ ] 이미지가 밝아짐

#### 테스트 케이스 2: 채도
- [ ] 채도 슬라이더 조정 (-50)
- [ ] 이미지 색상이 옅어짐

#### 테스트 케이스 3: 명암
- [ ] 명암 슬라이더 조정 (+50)
- [ ] 명암이 증가됨

#### 테스트 케이스 4: 색온도
- [ ] 색온도 슬라이더 조정 (+30)
- [ ] 이미지가 따뜻한 톤으로 변함

### 7. Frame (액자) 기능

- [ ] **프레임 타입 선택**
  - 검정 테두리
  - 흰색 테두리
  - 이중 테두리

- [ ] **크기 조정**
  - 슬라이더로 5px ~ 50px 조정
  - Canvas 크기가 프레임 크기만큼 증가

- [ ] **색상 변경**
  - 색상 선택기에서 다른 색 선택
  - 테두리 색이 변함

### 8. Signature (서명) 기능

#### 테스트 케이스 1: 텍스트 서명
- [ ] "텍스트" 탭 선택
- [ ] 서명 텍스트 입력 ("John Doe")
- [ ] 위치 선택 (우하단)
- [ ] 폰트 크기 설정 (28px)
- [ ] 색상 선택 (검정)
- [ ] "적용" 버튼 클릭
- [ ] Canvas 우하단에 "John Doe" 텍스트 표시

#### 테스트 케이스 2: 템플릿 서명
- [ ] "템플릿" 탭 선택
- [ ] 템플릿 선택 ("날짜")
- [ ] "적용" 버튼 클릭
- [ ] 현재 날짜가 표시됨

### 9. Mosaic (모자이크) 기능

- [ ] 모자이크 크기 설정 (15px)
- [ ] 영역 입력 (100,100,200,200,10)
  - x=100, y=100, width=200, height=200, pixelSize=10
- [ ] "적용" 버튼 클릭
- [ ] 지정된 영역이 모자이크 처리됨

### 10. Sticker (스티커) 기능

#### 테스트 케이스 1: 이모지
- [ ] "이모지" 카테고리 선택
- [ ] 이모지 버튼 클릭 (😊)

#### 테스트 케이스 2: 아이콘
- [ ] "아이콘" 카테고리 선택
- [ ] 아이콘 버튼 클릭 (✓)
- [ ] Canvas에 아이콘 추가됨

#### 테스트 케이스 3: 도형
- [ ] "도형" 카테고리 선택
- [ ] 도형 버튼 클릭 (●)
- [ ] Canvas에 도형 추가됨

### 11. 히스토리 기능

- [ ] **Undo (취소)**
  1. Crop 적용
  2. Filter 적용
  3. Undo 버튼 클릭
  4. Filter 제거되고 Crop만 남음

- [ ] **Redo (재실행)**
  1. 위 상태에서 Redo 버튼 클릭

- [ ] **Reset (초기화)**

  2. Reset 버튼 클릭

### 12. 종료 및 저장

  1. 편집 수행
  2. "적용" 버튼 클릭
  3. 모달 닫힘
  4. 이미지가 업데이트됨

- [ ] **취소**
  1. 편집 수행

## 🔧 디버깅 팁

### 1. 콘솔 로그 확인
```javascript
// 플러그인 상태 확인
console.log(editor.plugins.smartImageEditor);

// 현재 Canvas 상태
console.log(editor.plugins.smartImageEditor.canvas);

// 편집 상태
console.log(editor.plugins.smartImageEditor.editState);

// 히스토리
console.log(editor.plugins.smartImageEditor.state.history);
```

### 2. Canvas 검사
```javascript
// Canvas 이미지 데이터 확인
const canvas = editor.plugins.smartImageEditor.canvas;
console.log(canvas.width, canvas.height);
console.log(canvas.toDataURL());
```

### 3. 이벤트 추적
```javascript
// smartImageEdited 이벤트 리스닝
editor.on('smartImageEdited', (data) => {
    console.log('이미지 편집됨:', data);
});
```
